## âœ¨ Purpose

The goal was to convert an offline paper leaflet into a digital, shareable webpage.  
Many local users struggle with complex systems, so the design is kept minimal and practical.

The website allows users to:  
- View marriage bureau details  
- Fill out a Google Form  
- Contact through WhatsApp  
- Access everything in one mobile-friendly page
